How to create valid CSV file

File struture

  • CSV file should contains list of address and balance. Each entry should be in new line;
  • Address and balance need to be separeted by semicolon ;
  • CSV file should not have headers.

Address format

  • Ethereum address should be in hex format and starts with 0x prefix.
  • Application automaticly check validity of all addresses in the imported file

Balance format

  • Decimals should be separated by a comma , or dot ..
  • It is no need to declare a value with decimals precision, Eth ToolKit automaticly calculate number of zeros , that need to be add during transaction ( base on token decimals property )
